# Git Ubuntu

## Install  
```
sudo apt update
sudo apt install git
```

## Set Up
```
git config --global user.name "Heru Sularto"
git config --global user.email "herusularto@gmail.com"
```

untuk memeriksa semua konfigurasi:  
```
git config --list
```

untuk mengedit lewat terminal:
```
nano ~/.gitconfig
```
